**Implementation rules (all from RESEARCH, cross-checked against pitfalls):**

1. **Hit-test loop (Update step 1):** first registered `Enabled && Bounds.Contains(mp)` wins. Tracks `_hovered` and previous `_hovered` for transitions.

2. **Hover transitions (step 2):** call `OnHoverExit` on previous, `OnHoverEnter` on new, reset `_hoverTime = 0`, fire `OnHoverSound?.Invoke()` on enter (not on move within same widget).

3. **Cursor cache (step 3, Pitfall 1):** Track `private bool _wasHoveringClickable`. Only call `Mouse.SetCursor(MouseCursor.Hand/Arrow)` when `_hovered != null` state CHANGES. Never per-frame unconditionally.

4. **Focus navigation (step 4):** On `input.IsKeyPressed(Keys.Tab)`, advance `_focused` to next/prev `Enabled` widget (Shift-Tab reverses). `AdvanceFocus(bool backward)`: index-based wrap-around, skip `!Enabled` widgets (Pitfall 5). Null-safe when `_widgets.Count == 0`.

5. **Click dispatch (step 5):**
   - If `input.IsLeftClickPressed && _hovered != null && _hovered.Enabled`: set `_focused = _hovered`, call `_hovered.OnClick()`, fire `OnClickSound`, return `true`.
   - Else if `_focused != null && _focused.Enabled && (IsKeyPressed(Enter) || IsKeyPressed(Space))`: call `_focused.OnClick()`, fire `OnClickSound`, return `true`.
   - Else return `false`.
   - **Input gating (CONTEXT):** the `true` return is the contract — callers use it to skip their own click handling.

6. **Draw method:**
   - Draw focus outline around `_focused` if `_focused.Enabled`: a 2px-thick golden ring (`new Color(226,190,114)`) around `Bounds` inflated by 1px. Use `DrawOutline(sb, pixel, rect, color, thickness)` static helper (draws 4 edge rectangles).
   - Draw tooltip if `_hovered?.Tooltip != null && _hoverTime >= TooltipDelay`:
     - Measure `font.MeasureString(text)`.
     - Anchor: `_hovered.Bounds.Right + 8` (to the right of widget) and `_hovered.Bounds.Y`.
     - Clamp X to `[4, viewportW - tooltipW - 4]` (Pitfall 7 — mirror ChestScene:817).
     - Pad `rect` by 8px per side. Call `WidgetHelpers.DrawTooltipPanel(sb, pixel, rect)`.
     - Draw text on top with `sb.DrawString(font, text, textPos, Color.LightGoldenrodYellow)`.

7. **Constants:**
   - `private const float TooltipDelay = 0.5f;` — per CONTEXT (~500ms).
   - `private static readonly Color FocusOutline = new(226, 190, 114);` — matches existing golden.

8. **Logging:** log prefix `[UIManager]`. Single startup log in first `Register` call OK; otherwise silent.

**Thread safety:** not required — game loop is single-threaded (XML-doc it).

**Decision traceability:** implements D-01 (scope: all clicks through manager), D-04 (tooltip 500ms), D-06 (Tab/Shift-Tab + Enter/Space), Extra-3 (SFX hooks no-op), discretion "SetCursor cache" + "input gating" + "focus visual outline".

```csharp
using Microsoft.Xna.Framework.Graphics;
using Microsoft.Xna.Framework.Input;                       // NEW — for Mouse.SetCursor
using stardew_medieval_v3.UI.Widgets;                      // NEW

namespace stardew_medieval_v3.Core;

public abstract class Scene
{
    protected ServiceContainer Services { get; }

    /// <summary>
    /// Per-scene UI widget orchestrator. Scenes register widgets in LoadContent,
    /// call Ui.Update(dt, Services.Input) first in Update to consume widget clicks,
    /// and Ui.Draw(...) last in Draw. Auto-created so scenes can use it immediately.
    /// Cleared in UnloadContent (also restores cursor to Arrow — Pitfall 2).
    /// </summary>
    protected UIManager Ui { get; } = new UIManager();

    protected Scene(ServiceContainer services) { Services = services; }

    public virtual void LoadContent() { }
    public virtual void Update(float deltaTime) { }
    public virtual void Draw(SpriteBatch spriteBatch) { }

    /// <summary>
    /// Base UnloadContent clears widget registrations and restores cursor.
    /// Derived overrides SHOULD call base.UnloadContent() to inherit this cleanup.
    /// </summary>
    public virtual void UnloadContent()
    {
        Ui.Clear();
        Mouse.SetCursor(MouseCursor.Arrow);                // Pitfall 2: no stale Hand cursor
    }
}
```
