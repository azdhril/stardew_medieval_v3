# Quick Task 260416-10o: Inventory / Chest / Shop HUD Visual Redesign — Summary

One-liner: Swapped placeholder `Color.DarkSlateGray` / `new Color(60,40,30)` / `new Color(78,58,44)` chrome on the three HUDs (Inventory, Chest, Shop) for pixel-art 9-slice textures via a new `NineSlice` helper + shared `UITheme` service registered on `ServiceContainer`.

## Files Changed (9 total: 2 new + 7 modified)

### New
- `src/UI/NineSlice.cs` — Stateless static helper. `Draw(SpriteBatch, Texture2D, Rectangle, Insets, Color?)` carves texture into 9 regions; `DrawStretched(...)` for fixed/stretched icons; `Insets.Uniform(int)` factory.
- `src/UI/UITheme.cs` — Owns all 15 UI Texture2D instances + per-texture 9-slice metadata. Loads via `File.OpenRead` + `Texture2D.FromStream` (mirrors existing `InventoryGridRenderer` precedent). Idempotent `LoadContent` — safe to call from any overlay scene. Fallback solid-color texture on any file error with `[UITheme] Failed to load ...` log line.

### Modified
- `src/Core/ServiceContainer.cs` — Added `public UITheme? Theme { get; set; }` slot after `Hotbar`.
- `src/UI/InventoryGridRenderer.cs` — Added `private UITheme? _theme` field + `SetTheme(UITheme)` injector. `DrawEquipment` now draws a `UI_Panel_Thumbnail` 9-slice frame around the silhouette when theme is set (skipped gracefully when null).
- `src/Scenes/InventoryScene.cs` — Lazy-init `Services.Theme` in `LoadContent`. Panel bg `Color.DarkSlateGray * 0.9f` → `UI_Pane_Popup` 9-slice. Title text → centered inside `UI_Panel_Title` plaque overlapping the top edge. 1px divider → `UI_Image_Deco` stretched into the same rect so equip/grid offsets don't shift.
- `src/Scenes/ChestScene.cs` — Lazy-init theme. Panel bg `new Color(35,28,24)*0.97f` + black outline → `UI_Pane_Popup`. Title `panelX+18, panelY+12` → centered inside `UI_Panel_Title` plaque at `panelX+12, panelY-8` (220x32). `DrawButton` → `UI_Common_Btn` 9-slice (outline + fill removed; text centering preserved). `GetButtonRect`, `HitTestButton`, `ExecuteButton`, and all drag/double-click/bulk-move code untouched.
- `src/Scenes/ShopOverlayScene.cs` — Lazy-init theme in `LoadContent`; passes `Services.Theme!` into new `ShopPanel.Draw` signature.
- `src/UI/ShopPanel.cs` — `Draw(sb, font, pixel)` → `Draw(sb, font, pixel, UITheme theme)`. Panel outline + fill → `UI_Pane_Popup` 9-slice. "Shop" header → centered in `UI_Panel_Title` plaque (160x30 at `PanelX+16, PanelY-8`). Gold label → wrapped in `UI_Panel_Currency` pouch sized to text. Close X → `UI_BtnIcon_x` stretched into `_closeRect` (no text "X" needed). `DrawTab` (text+pixel signature) → uses `theme.TabOn`/`theme.TabOff` 9-slice. Row hover → `UI_ListItem_Hover` 9-slice. Qty stepper minus/qtyLabel/plus → `UI_Square_Btn` 9-slice (qtyLabel dimmed via `Color.White * 0.6f` tint). Action button → `UI_Btn_yellow_small` 9-slice (full white when enabled, `* 0.4f` when disabled). Divider under tabs → `UI_Image_Deco` stretched to 4px height. Scrollbar track/thumb left as thin solid rects (plan exception). All 11 cached rects (`_panelRect`, `_buyTabRect`, `_sellTabRect`, `_closeRect`, `_rowRects[]`, `_actionBtnRects[]`, `_qtyMinusRects[]`, `_qtyLabelRects[]`, `_qtyPlusRects[]`, `_scrollTrackRect`, `_scrollThumbRect`) retain identical coordinates. `UpdateLayoutCache`, `Update`, `TryBuy`, `TrySell`, `EnsureRowQtySized`, `GetMaxQty`, `ComputeStackHeadroom`, `IndexOfNthFilledSlot`, `IsActionEnabled`, `CanAddOne`, `GetRowCount` all untouched. `PanelFill` + `Bevel` constants retained per plan (Bevel still used for scrollbar track).
