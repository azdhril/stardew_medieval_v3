# Phase 04.1: Shop UX & Save Gap Closure — Context

**Gathered:** 2026-04-13
**Status:** Ready for planning

<decisions>

## Shop UX Model (Stardew Valley style)

**D-01. No persistent "selected item" concept.**
Remove `_selectedIndex` as a driver of anything user-visible. There is no highlighted/selected row that persists across interactions. Hover state is fine (visual feedback), but selection is not a first-class UX concept.

Why: Stardew Valley's shop has no "selected item" — you click Buy on any row, adjust qty, click again. Selection-driven scrolling created the regression being fixed and adds cognitive overhead for no benefit.

**D-02. Scroll is pure wheel / manual.**
`_scrollOffset` is modified only by:
- Mouse wheel (clamped to valid range)
- (Optional) a scrollbar drag if present

Remove the follow-selection block in `UpdateLayoutCache` that resets `_scrollOffset` to keep `_selectedIndex` visible. Scroll never jumps on its own.

**D-03. Input is 100% mouse.**
No keyboard navigation. Remove (or ignore) Up/Down/Enter/+/- keyboard handlers inside the shop panel. Escape to close stays (standard close affordance).

**D-04. Per-row quantity stepper (Stardew-like).**
Each row has its own `-` / `qty` / `+` controls and its own `Buy` / `Sell` action button. Click flow:
- Click `+`/`-` on a row → adjusts that row's qty
- Click `Buy`/`Sell` on a row → executes transaction for that row's qty
- qty state is per-row; no global "selected qty"

Minimum qty = 1. Maximum = min(affordable, stack cap, shop stock if applicable).

**D-05. Tabs and Close are mouse-only too.**
Buy / Sell tabs switch on click. Close `X` and click-outside-panel close the shop. All click paths log via `[ShopPanel]`.

## Save Round-Trip (unchanged from original scope)

**D-06. Gold + quest state + inventory must persist outside Farm sleep.**
Add a shared `GameStateSnapshot.SaveNow(Services)` entry point and trigger it on:
- FarmScene.OnDayAdvanced (existing)
- ShopOverlayScene close (new)
- GameplayScene F5 hotkey (new — manual save)

</decisions>
