**4. Create Combat/EnemyAI.cs** -- FSM logic per D-19:
- Namespace: `stardew_medieval_v3.Combat`
- Enum `EnemyState { Idle, Chase, Attack, Return }`
- Class `EnemyAI` with fields: `EnemyState State`, `Vector2 SpawnPosition`, `float _attackTimer`.
- Method `void Update(float deltaTime, Vector2 enemyPos, Vector2 playerPos, EnemyData data)`:
  - Calculate `float distToPlayer = Vector2.Distance(enemyPos, playerPos);`
  - State transitions per D-19:
    - Idle: if distToPlayer < data.DetectionRange => Chase
    - Chase: if distToPlayer < data.AttackRange => Attack; if distToPlayer > data.DetectionRange * 1.5f => Return
    - Attack: if distToPlayer > data.AttackRange * 1.2f => Chase; decrement _attackTimer, when <= 0 flag attack ready and reset to data.AttackCooldown
    - Return: move toward SpawnPosition; if distance to spawn < 4f => Idle
  - For ranged enemies (DarkMage): Chase state tries to maintain data.AttackRange distance (move AWAY if closer than AttackRange * 0.6f, approach if farther than AttackRange).
- Method `Vector2 GetMoveDirection(Vector2 enemyPos, Vector2 playerPos)`: returns unit vector based on State (Chase: toward player; Return: toward SpawnPosition; Attack for ranged: maintain distance; else Vector2.Zero).
- Property `bool IsAttackReady`: true when _attackTimer fires (reset each attack).
