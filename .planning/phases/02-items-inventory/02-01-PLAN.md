**Create Inventory/InventoryManager.cs** in namespace `stardew_medieval_v3.Inventory`:
- `public const int SlotCount = 20;`
- `public const int HotbarSize = 8;`
- Private field: `private readonly ItemStack?[] _slots = new ItemStack?[SlotCount];`
- `public int ActiveHotbarIndex { get; private set; } = 0;`
- `public string? WeaponId { get; private set; }`
- `public string? ArmorId { get; private set; }`
- `public event Action? OnInventoryChanged;`
- `public ItemStack? GetSlot(int index)` -- returns `_slots[index]` with bounds check.
- `public int TryAdd(string itemId, int quantity = 1)` -- Two-pass: (1) fill existing stacks of same itemId up to `ItemRegistry.Get(itemId).StackLimit`, (2) place remainder in first empty slot. Return remaining quantity (0 = all added). Call `OnInventoryChanged?.Invoke()` if anything changed.
- `public ItemStack? RemoveAt(int index)` -- Remove and return the stack at index, set slot to null, invoke event.
- `public void MoveItem(int fromSlot, int toSlot)` -- Swap contents of two slots. If same itemId and stackable, merge up to StackLimit and leave remainder in fromSlot. Invoke event.
- `public void SetActiveHotbar(int index)` -- Clamp 0-7, set `ActiveHotbarIndex`.
- `public ItemStack? GetActiveHotbarItem()` -- returns `GetSlot(ActiveHotbarIndex)`.
- `public bool TryEquip(int slotIndex)` -- If slot has a Weapon item (via `ItemRegistry.Get()`), set `WeaponId` to that item's Id and remove from slot. If Armor, set `ArmorId`. Swap with currently equipped item (put old equipment back in the slot). Return true if equipped. Invoke event.
- `public bool TryUnequip(string equipType, int targetSlot)` -- If targetSlot is empty or can stack, move weapon/armor back to inventory. equipType is "weapon" or "armor".
- `public void LoadFromState(GameState state)` -- Populate `_slots` from `state.Inventory`, set `WeaponId = state.WeaponId`, `ArmorId = state.ArmorId`, `ActiveHotbarIndex` from first non-null in `state.HotbarSlots` or 0.
- `public void SaveToState(GameState state)` -- Write `_slots` (non-null only) to `state.Inventory`, set `state.WeaponId/ArmorId`, populate `state.HotbarSlots` from first 8 slot ItemIds.
