**Create Entities/ItemDropEntity.cs** in namespace `stardew_medieval_v3.Entities`:
- Extends `Entity` from `stardew_medieval_v3.Core`.
- Constructor: `public ItemDropEntity(string itemId, int quantity, Vector2 spawnPosition, SpriteAtlas atlas)`
  - Store `_itemId`, `_quantity`, `_atlas`.
  - Set `Position = spawnPosition`.
  - Store `_startPos = spawnPosition`.
  - Generate random lateral bounce offset: `_bounceOffset = new Vector2(random.Next(-8, 9), random.Next(-4, 5))` using a static Random.
  - Set `_isBouncing = true`, `_bounceTimer = 0f`.
  - Look up sprite rect: `_spriteRect = atlas.GetRect(ItemRegistry.Get(itemId)?.SpriteId ?? "")`.
  - Per D-08: Uses the real item sprite, not a generic sack.
- Public properties:
  - `public string ItemId => _itemId;`
  - `public int Quantity { get; set; }` -- initialized from constructor quantity. Mutable because partial pickup may reduce it.
  - `public bool IsCollected { get; private set; } = false;`
- Constants:
  - `private const float MagnetRange = 56f;` (per D-09: ~48-64px)
  - `private const float PickupRange = 8f;`
  - `private const float MaxMagnetSpeed = 200f;`
  - `private const float BounceDuration = 0.4f;`
  - `private const float BounceHeight = 12f;`
- `public void UpdateWithPlayer(float deltaTime, Vector2 playerPos, InventoryManager inventory)`:
  - If `IsCollected`, return immediately.
  - If `_isBouncing`:
    - Increment `_bounceTimer += deltaTime`.
    - Calculate `t = MathHelper.Clamp(_bounceTimer / BounceDuration, 0f, 1f)`.
    - Parabolic arc: `yOffset = -BounceHeight * 4f * t * (1f - t)` (peaks at t=0.5).
    - `xOffset = _bounceOffset.X * t`.
    - `Position = _startPos + new Vector2(xOffset, yOffset + _bounceOffset.Y * t)`.
    - If `t >= 1f`, set `_isBouncing = false`.
    - Per D-10: Bounce animation on spawn.
    - Return (no magnetism during bounce).
  - Calculate distance to player: `float dist = Vector2.Distance(Position, playerPos);`
  - If `dist <= PickupRange`:
    - `int remaining = inventory.TryAdd(_itemId, Quantity);`
    - If `remaining == 0`: set `IsCollected = true` (fully picked up).
    - Else if `remaining < Quantity`: set `Quantity = remaining` (partial pickup, per D-13: rest stays on ground).
    - Else: do nothing (inventory completely full, item stays, per D-13).
    - Return.
  - If `dist <= MagnetRange`:
    - Per D-09: quadratic ease-in acceleration toward player.
    - `float t = 1f - (dist / MagnetRange);` (0 at edge, 1 near center)
    - `float speed = MathHelper.Lerp(40f, MaxMagnetSpeed, t * t);`
    - `Vector2 direction = Vector2.Normalize(playerPos - Position);`
    - `Position += direction * speed * deltaTime;`
- Override `Draw(SpriteBatch spriteBatch)`:
  - If `IsCollected`, return.
  - Draw the item icon at Position using `_atlas.Texture` and `_spriteRect`.
  - Size: 16x16 pixels at world scale. Use `spriteBatch.Draw(_atlas.Texture, new Rectangle((int)Position.X - 8, (int)Position.Y - 8, 16, 16), _spriteRect, Color.White);`
  - Add a subtle bobbing animation for idle items (not bouncing): `float bob = (float)Math.Sin(totalTime * 3f) * 1.5f;` offset to Y position. Use a private `_totalTime` field incremented in UpdateWithPlayer.
