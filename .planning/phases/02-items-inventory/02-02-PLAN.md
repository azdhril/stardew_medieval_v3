  - If click hits a valid slot (0-19):
    - If `_selectedSlot == -1`: set `_selectedSlot = hitSlot` (select it).
    - Else: call `_inventory.MoveItem(_selectedSlot, hitSlot)`, then `_selectedSlot = -1` (deselect).
    - Return true.
  - Return false if click missed all slots.

  - Per D-01: Grid estilo Stardew Valley com 20 slots em caixinhas.
  - Layout: 5 columns x 4 rows. slotSize=36, padding=4.
  - For each slot 0-19:
    - Calculate x, y from col/row + offset.
    - Draw slot texture: `_slotSelected` if `i == _selectedSlot`, else `_slotNormal`.
    - Get `stack = _inventory.GetSlot(i)`. If not null:
      - Draw item icon from `_atlas.GetRect(ItemRegistry.Get(stack.ItemId).SpriteId)` inside slot (2px inset).
      - Per D-03: Draw rarity tint overlay. Get item's Rarity from `ItemRegistry.Get(stack.ItemId)?.Rarity`. Colors: `Uncommon -> new Color(50, 205, 50) * 0.3f` (green glow), `Rare -> new Color(255, 215, 0) * 0.3f` (gold glow), `Common -> no overlay`. Draw tint using `_pixel` over the full slot rect.
      - If `stack.Quantity > 1`, draw quantity string at bottom-right of slot in Color.White.
