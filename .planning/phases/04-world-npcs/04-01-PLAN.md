## STRIDE Threat Register (ASVS L1)

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-01 | Tampering | savegame.json Gold/QuestState | accept | Single-player local game; tamper = self-cheat. No server. Documented acceptance. Clamp QuestState to [0,2] in v5 migration prevents enum cast OOB. |
| T-04-02 | Tampering | savegame.json SaveVersion | mitigate | Migration chain is monotonic (each branch bumps version); unknown future version still loads current-version defaults via JSON leniency. Clamp in v5 migration ensures QuestState stays in enum range. |
| T-04-03 | Denial of Service | TileMap on malformed Triggers group | mitigate | Use `layer.objects ?? Array.Empty<TiledObject>()` and `obj.name ?? ""` to tolerate missing XML attributes. Matches existing LoadCollisionObjects defensive pattern. |
| T-04-04 | Information Disclosure | MainQuest state event | accept | Internal process event; no IPC. No mitigation needed. |
| T-04-05 | Elevation of Privilege | Gold underflow via TrySpendGold | mitigate | `TrySpendGold` rejects negative amounts and amounts exceeding Gold; returns bool for caller to branch on. Gold clamped to >=0 in SetGold. |
</threat_model>
