public abstract class Entity
{
    public Vector2 Position { get; set; }
    public Vector2 Velocity { get; set; }
    public Direction FacingDirection { get; set; } = Direction.Down;
    public float HP { get; set; } = 100f;
    public float MaxHP { get; set; } = 100f;
    public bool IsAlive => HP > 0;
    protected Texture2D? SpriteSheet { get; set; }
    protected int FrameWidth { get; set; }
    protected int FrameHeight { get; set; }
    protected int FrameIndex { get; set; }
    protected float AnimationTimer { get; set; }
    protected float FrameTime { get; set; } = 0.15f;
    public virtual Rectangle CollisionBox { get; }
    public virtual void Update(float deltaTime) { }
    public virtual void Draw(SpriteBatch spriteBatch) { }
}

   - Namespace: `stardew_medieval_v3.Entities`
   - `public class DummyNpc : Entity`
   - Construtor recebe `Texture2D pixel` (1x1 branco, reusado do TestScene) e `Vector2 startPosition`
   - Armazena o pixel em SpriteSheet (herdado, protected setter)
   - Define dimensoes visuais: FrameWidth=16, FrameHeight=24 (tamanho de um NPC pixel art)
   - Define Position = startPosition
   - Override CollisionBox: retorna Rectangle centrado na base da entidade (12x8 pixels nos pes)
   - Override Update(float deltaTime): movimento simples de patrulha horizontal (pace back and forth).
     Usa um campo privado `_paceDirection` (1 ou -1) e `_paceTimer`. A cada 2 segundos inverte direcao.
     Move Position.X += _paceDirection * 30f * deltaTime (30 pixels/segundo, bem lento).
     Atualiza FacingDirection conforme direcao do movimento.
   - Override Draw(SpriteBatch spriteBatch): desenha um retangulo colorido (Color.ForestGreen) usando
     SpriteSheet (pixel texture) com as dimensoes FrameWidth x FrameHeight na Position.
     Desenha tambem o CollisionBox como retangulo semi-transparente vermelho para debug visual.
   - Docstrings em todos os membros publicos (seguindo convencao do projeto).

3. Modificar `src/Scenes/TestScene.cs`:
   - Adicionar `using stardew_medieval_v3.Entities;` no topo
   - Adicionar campo `private DummyNpc _npc = null!;`
   - Em LoadContent(), apos carregar _font, instanciar:
     `_npc = new DummyNpc(_pixel, new Vector2(viewport.Width / 2f, viewport.Height / 2f + 60f));`
     (Nota: usar Services.GraphicsDevice.Viewport para pegar dimensoes)
   - Em Update(), chamar `_npc.Update(deltaTime);` ANTES do check da tecla B
   - Em Draw(), dentro do bloco spriteBatch.Begin/End existente, chamar `_npc.Draw(spriteBatch);`
     DEPOIS do DrawString (para o NPC aparecer sobre o fundo)
   - Adicionar label "DummyNpc (Entity test)" abaixo do texto existente usando _font, cor Color.LightGreen
